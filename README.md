# StopChurn WordPress Connector

The **StopChurn WP Connector** plugin connects your WordPress site to [StopChurn](https://stopchurn.com).
It automatically sends form submissions (currently supports **Contact Form 7**) to StopChurn so you can capture and track leads, messages, or customer interactions directly from your site.

---

## Installation

1. Download this repository ZIP.
2. In your WordPress admin, go to **Plugins → Add New → Upload Plugin**.
3. Upload the ZIP file, click **Install Now**, then **Activate**.

---

## Configuration

1. After activation, go to **Settings → StopChurn Connector**.
2. Enter your StopChurn API key and Brand details.
3. Save changes.
